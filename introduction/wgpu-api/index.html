    Wgpu Api

    What is Wgpu

    wgpu is a safe and portable graphics library for Rust based on the WebGPU API. It is suitable for general purpose graphics and compute on the GPU. Applications using wgpu run natively on Vulkan, Metal, DirectX 12, and OpenGL ES; and browsers via WebAssembly on WebGPU and WebGL2.
